Pourquoi devenir développeuse web?

Un site web est aujourd'hui primordial pour tout business.
C'est l'image d'une entreprise, un outil de mise en relation avec les clients, une application fonctionnelle.

Pour se démarquer, les demandes sont toujours plus exigeantes : il faut un site plus performant, plus original, créatif et fluide. Pour répondre à ces besoins, les technologies évoluent au quotidien. Les opportunités à venir dans le milieu sont donc nombreuses.

C'est ce dynamisme qui m'a donné envie de devenir développeuse web.

Vers une spécialisation Front-end

J'ai une grande attirance pour le Front-end : reproduire fidèlement une maquette et veiller à une interface utilisateur fluide me passionnent. Je souhaite offrir la meilleure expérience possible au travers de belles animations et d'un design attrayant.

Le Front-end est primordial, car un site web est l'identité même d'une entreprise. C'est pourquoi je réaliserai avec rigueur et minutie toute les missions qui me seront confiées.

De nature curieuse, j'aime apprendre et je perfectionne mon code de jour en jour. J'ajoute régulièrement mes projets sur ce portfolio.
